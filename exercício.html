Instruções: responda cada nível criando o HTML pedido (use <ol> para listas ordenadas e <ul> para listas não ordenadas). As questões são simples e usam o tema BYD — carros elétricos.

Lista Ordenada

Contexto: passos para preparar um test drive de um carro elétrico BYD.

Questão 1

Lista Não Ordenada

Questão 2
